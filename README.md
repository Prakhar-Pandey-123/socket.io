<h1>Basic Real-Time Chat App</h1>

<p>
A simple real-time chat application built using React and Socket.IO.
This project demonstrates real-time communication between client and server using WebSockets.
</p>

<h2>Tech Stack</h2>
<ul>
  <li>Frontend: React, Socket.IO Client , Typescript ,Tailwind </li>
  <li>Backend: Node.js, Express, Socket.IO</li>
</ul>

<h2>Features</h2>
<ul>
  <li>Real-time messaging</li>
  <li>Socket connection handling using useEffect</li>
  <li>State management with useState</li>
  <li>Dynamic UI updates on new messages</li>
</ul>

<h2>How to Run</h2>
<pre>
Backend:
cd server
npm install
node index.js

Frontend:
cd cleint
npm install
npm run dev
</pre>

<p>
This project was built for learning real-time communication and understanding React re-render behavior.
</p>
